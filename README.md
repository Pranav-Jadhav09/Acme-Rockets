## Run Locally

To run **Omnifood**, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/Pranav-Jadhav09/omnifood-website.git
```

Windows:

```bash
git clone https://github.com/Pranav-Jadhav09/omnifood-website.git
```
